To compile this demo, edit the file

    chrono_proj_config.vsprops 
    
that is contained in this directory, and
modify the field with the setting for the 
directory where you have your copy of the 
Irrlicht SDK:
  

....

<UserMacro
		Name="IrrlichtDir"
		Value="[YOUR IRRLICHT SDK DIRECTORY HERE!!!]"
/>

.....
